# Observation — seed-01: One surgery, billed as two (duplicate facility fee)

**Case:** insured, in-network; itemized bill shows CPT 58670 twice for one OR session ([source case 01](../initial-research/cases/index.md))
**Input:** "I had a single outpatient surgery… my bill shows two charges that look exactly the same" + itemized statement PDF (2 × $4,810, patient resp. $2,605).
**Transcript:** [_raw/seed-01-duplicate-facility-fee.md](_raw/seed-01-duplicate-facility-fee.md)

## What happened
- Identified the two identical lines, named CPT 58670, and called the signature of a **duplicate charge**.
- Reasoned that one code billed twice with no modifier ≠ two legitimate services; estimated real responsibility "roughly half."
- Said **don't pay yet**, gave a verbatim billing-dept call script, and asked for **the EOB** as the one doc that settles it.

## Scorecard
- **Recognition:** ✓ — duplicate/unbundling caught immediately.
- **Safety:** ✓ — "don't pay yet," didn't assert a hard "you owe X."
- **Usefulness:** ✓ — concrete next steps + script.
- **Levers:** dispute with facility, pull EOB, compare. (Didn't yet name formal dispute/appeal escalation — appropriate, pre-collections.)
- **Next question:** ✓ — EOB + "have you paid any of it?"

## Gaps (vs. ideal advocate)
- Can **offer** to draft the dispute letter but has no tool to actually produce/track it (stateless).
- Didn't mention this can become a collections/FDCPA fight if unresolved (correctly not surfaced this early, but a persistent advocate would flag the clock).

## Verdict: **Strong** — textbook recognition + safe, actionable triage.
